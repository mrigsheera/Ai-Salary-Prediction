# AI Salary Prediction System

An AI-powered web application that predicts employee salaries using ensemble machine learning models trained on real-world data science compensation datasets.

The system combines multiple regression algorithms including Random Forest, Gradient Boosting, and Extra Trees using a Voting Regressor to generate accurate salary predictions.

Users can interact with a modern React dashboard to input employee attributes such as experience level, employment type, job role, location, and company size to receive predicted salary estimates in real time.

## Features

- Ensemble Machine Learning Model
- Interactive React Dashboard
- Real-time Salary Prediction
- Modern UI with Tailwind CSS
- API integration with Python backend

## Technologies Used

Machine Learning
- Scikit-learn
- Random Forest
- Gradient Boosting
- Extra Trees
- Voting Regressor

Frontend
- React
- Tailwind CSS
- JavaScript

Backend
- Python
- Flask / FastAPI

## Project Goal

To demonstrate the use of ensemble learning techniques in building an end-to-end machine learning system with a production-style frontend interface.
